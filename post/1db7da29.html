1. Cmd Markdown 简明语法手册

1.1. 斜体和粗体

使用 * 和 ** 表示斜体和粗体。

示例：

这是 斜体，这是 粗体。

1.2. 分级标题

使用 === 表示一级标题，使用 — 表示二级标题。

示例：

你也可以选择在行首加井号表示不同级别的标题 (H1-H6)，例如：# H1, ## H2, ### H3，#### H4。

1.3. 外链接

使用 描述 为文字增加外链接。

示例：

这是去往 本人博客 的链接。

1.4. 无序列表

使用 *，+，- 表示无序列表。

示例：

  • 无序列表项 一
  • 无序列表项 二
  • 无序列表项 三

1.5. 有序列表

使用数字和点表示有序列表。

示例：

  1. 有序列表项 一
  2. 有序列表项 二
  3. 有序列表项 三

1.6. 文字引用

使用 > 表示文字引用。

示例：

野火烧不尽，春风吹又生。

1.7. 行内代码块

使用 代码 表示行内代码块。

示例：

让我们聊聊 html。

1.8. 代码块

使用 四个缩进空格 表示代码块。

示例：

1.9. 插入图像

使用 ![img](图片链接地址) 插入图像。

示例：

我的头像

2. Cmd Markdown 高阶语法手册

2.1. 内容目录

在段落中填写 [TOC] 以显示全文内容的目录结构。

[TOC]

2.2. 标签分类

在编辑区任意行的列首位置输入以下代码给文稿标签：

标签： 数学 英语 Markdown

或者

Tags： 数学 英语 Markdown

2.5.3.4. 甘特图

甘特图内在思想简单。基本是一条线条图，横轴表示时间，纵轴表示活动（项目），线条表示在整个期间上计划和实际的活动完成情况。它直观地表明任务计划在什么时候进行，及实际进展与计划要求的对比。
